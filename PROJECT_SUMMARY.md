## 📊 技术栈

### 后端（数据抓取）
- Python 3
- 标准库：urllib, xml.etree, csv, json
- 无需额外依赖

### 前端
- HTML5
- CSS3（变量、Grid、Flexbox、动画）
- Vanilla JavaScript（ES6+）
- 无框架依赖

### 自动化
- GitHub Actions
- Cron 定时任务

### 部署
- Vercel（推荐）
- 或 Netlify / GitHub Pages
