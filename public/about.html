About Book Club Event Finder

Connecting readers with book clubs and literary events across North America

Our Mission

Book Club Event Finder was created to solve a simple problem: it's hard to discover book club events and reading groups in your area. While public libraries offer hundreds of amazing literary programs, finding them requires visiting multiple library websites individually.

Our mission is to centralize and simplify the discovery of book club events by aggregating programs from libraries across the United States and Canada into one searchable, easy-to-use platform. We believe that everyone deserves access to vibrant reading communities, whether online or in person.

How It Works

Book Club Event Finder uses automated data aggregation to collect event information from public library RSS feeds:

  • Daily Updates: Every day, our system automatically fetches the latest events from 66+ library RSS feeds using GitHub Actions
  • Smart Filtering: Events are parsed, deduplicated, and categorized (online vs. in-person, by location, by date)
  • Automatic Cleanup: Past events are automatically removed to keep listings fresh and relevant
  • Static Site Generation: Popular filter views (Today, Tomorrow, This Month, etc.) are pre-rendered daily for fast SEO-friendly pages
  • Real-time Search: Users can search by book title, author, library, or topic with instant results

Note: We aggregate publicly available event data. For registration and attendance policies, please refer to each library's official page.

Technology Stack

This platform is built with modern web technologies to ensure fast performance and reliable daily updates:

Python JavaScript GitHub Actions Jinja2 Templates RSS Feed Parsing JSON-LD Schema Google Analytics Static Site Generation

The entire platform is automated and runs serverlessly using GitHub Pages and GitHub Actions, ensuring zero downtime and minimal maintenance costs.

Library Coverage

We currently aggregate events from major public library systems including:

  • California: San Francisco, San José, Oakland, Palo Alto, Pleasanton, Alameda County, Contra Costa County, San Mateo County, Santa Clara County
  • New York: New York Public Library system and branches
  • Other States: Libraries across Washington, Oregon, Texas, Illinois, Massachusetts, and more
  • Canada: Select library systems in major Canadian cities

Our coverage continues to grow. If you'd like to suggest a library system to add, please reach out!

Privacy & Data

Your Privacy Matters: Book Club Event Finder does not collect personal information. We use Google Analytics to understand aggregate usage patterns (page views, popular searches) to improve the service.

Data Source: All event data is sourced from publicly available library RSS feeds. We do not modify event content; we simply aggregate and present it in a searchable format.

Registration: When you click to register for an event, you are directed to the library's official registration page. We do not handle registration ourselves.
