Frequently Asked Questions

Everything you need to know about finding and joining book club events

General Information

What is Book Club Event Finder?

Book Club Event Finder is a free service that aggregates book club and reading events from over 66 public libraries across North America. We collect over 1400+ events and update our database daily, making it easy to discover book clubs, author talks, and literary discussions in your area or online.

How often is the event data updated?

Our event database is automatically updated every day via GitHub Actions. We fetch fresh data from library RSS feeds daily to ensure you see the most current events. Past events are automatically removed from the listings.

Which libraries are included in this database?

We currently aggregate events from 66 public library systems across the United States and Canada, including major systems like San Francisco Public Library, San José Public Library, Oakland Public Library, New York Public Library, and many more. The full list of libraries can be seen in our State/Province filter on the main page.

Finding Events

How do I find book club events near me?

You can find local book club events by:

  • Using the search bar to enter your city or library name
  • Clicking the 'Filters' button and selecting your state or province
  • Browsing the 'Bay Area' quick filter if you're in the San Francisco Bay Area
  • Filtering by 'In Person' to see only physical events at libraries near you

Are there online book clubs available?

Yes! Many libraries offer online book club events via Zoom or other platforms. To find online events, click the 'Filters' button and select 'Online Only' under Event Type. Online events are marked with an 'ONLINE' badge and are accessible from anywhere with an internet connection.

Can I filter events by date?

Yes! Use our quick filter buttons to easily find events:

  • 'Today' shows events happening today
  • 'Tomorrow' shows next-day events
  • 'This Month' displays all events in the current month
  • 'Next Month' shows upcoming events

You can also use the search bar to find events by book title, author, or topic.

Registration & Attendance

How do I register for a book club event?

To register for an event:

  1. Click on the event name or 'View' button to see full details
  2. On the event details page, click the 'Register for this Event' button
  3. You'll be directed to the library's official registration page where you can sign up

Each library manages its own registration process.

Do I need a library card to attend?

Requirements vary by library and event. Many public library events are free and open to all community members, while some may require a library card for registration. Check the specific event details on the library's registration page for attendance requirements.

Are book club events free?

Most public library book club events are completely free to attend. Libraries offer these programs as a community service. However, you may need to obtain the book being discussed through purchase, library checkout, or digital lending. Check individual event details for any specific requirements.

Event Types

What types of book clubs are available?

Libraries offer diverse book clubs including:

  • General fiction and non-fiction clubs
  • Genre-specific clubs (mystery, science fiction, romance)
  • Classics discussion groups
  • Young adult book clubs
  • Children's book clubs
  • Author-led discussions
  • Multicultural reading groups
  • Specialized topics like banned books, poetry, or graphic novels
