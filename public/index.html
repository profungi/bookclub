What is Book Club Event Finder?

Book Club Event Finder is a free service that helps you discover book clubs, reading groups, and literary events from public libraries across the United States and Canada. We aggregate over 1400 events from 66+ library systems and update our listings daily.

How to use this site:

  • Use the search bar to find events by book title, author, library, or topic
  • Filter by date: Today, Tomorrow, This Month, or Next Month
  • Filter by type: Online events or In-Person events
  • Filter by location: Browse by state, province, or Bay Area libraries
  • Click any event to view details and register through the library's website
